V1.0.4 / 3-Feb-2015

Main Changes

  • update for the latest version of Standard Peripheral Library

 

Development Toolchains

  • EWARM v7.30
  • MDK-ARM v5.12
  • TrueSTUDIO v5.1.1

V1.0.3 / 30-April-2013

Main Changes

  • added project settings for 32L152C-DISCOVERY board
  • update for the latest version of Touch-Sensing Driver and Standard Peripheral Library

Development Toolchains

  • IAR EWARM version v6.50.3
  • Keil MDK-ARM  v4.60
  • Attolic TrueSTUDIO v4.0
  • TASKING VX-toolset for ARM v4.4

Main Changes

  • Initial release of the STM32L-Discovery Temperature sensor fw example

 

Development Toolchains

  • Keil MDK-ARM 4.21
  • Attolic TrueSTUDIO V2.1.0
